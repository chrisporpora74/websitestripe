# Astral Light Healings — Stripe Server

Express backend that creates Stripe Payment Intents for the shop checkout.

## Deploy to Railway
1. Push this folder to a GitHub repository
2. Connect the repo to Railway
3. Add environment variable: STRIPE_SECRET_KEY = your Stripe secret key
4. Railway will deploy automatically and give you a public URL

## Environment Variables
- STRIPE_SECRET_KEY — your Stripe secret key (from dashboard.stripe.com)
